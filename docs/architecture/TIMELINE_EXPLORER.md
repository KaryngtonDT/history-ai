# Timeline Explorer

Chronological view of learning events aggregated into the Second Brain.

## View

Month-by-month concept milestones (Jan Docker → Feb Symfony → …).

Drill-down: all videos, conversations, exercises, missions for selected period.

## Data sources

Memory timeline events + teaching progress + mentor mission completions + graph first-seen dates.

## API

`GET /api/shadow/brain/timeline?from=&to=`
