# Relationship Timeline

Timeline entries are `RelationshipObservation` records.

## Types

- `started_topic`
- `preference_changed`
- `trait_confirmed`
- `trait_inferred`
- `shared_reference`
- `challenge_adjusted`
- `communication_shift`
- `reset`

Each entry includes `label`, `detail`, and `recordedAt` for the Relationship settings UI.

The timeline answers: **why does Shadow behave this way right now?**
