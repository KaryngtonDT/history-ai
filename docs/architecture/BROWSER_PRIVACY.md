# Browser Privacy

## Principles

1. No automatic content reading
2. Explicit per-site permissions
3. Revocable at any time
4. Full activity journal with explainability

## Permission center

`/settings/shadow/browser` — per-host toggles (overlay, voice, watch, second brain, learning, clipboard, screen capture).

## API

- `GET /api/shadow/browser/permissions`
- `PUT /api/shadow/browser/permissions`
