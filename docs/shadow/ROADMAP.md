| Sprint | Capability | Priority |

### Sprint 67 highlights

- Unified **Knowledge Explorer** — search "Docker" → videos, PDFs, conversations, missions, graph
- **Knowledge Diff** — before watch: new vs known concepts, redundancy %
- **Knowledge Merge** — duplicate concepts across PDFs/sources fused deterministically
- **Knowledge Sources** — jump back to exact learning moment
- Timeline, bookmarks, personal notes (user-owned, separate from auto-knowledge)
- Stretch: **Concept Evolution**, **Learning Connections Heatmap**

- Billing

Previous plan (S67 platform hardening, S68 public API, …) is **superseded** by this roadmap.
