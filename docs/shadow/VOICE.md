# Shadow — Voice

Voice Studio (Sprint 58) supplies personas and presets. Teaching (S63) adds **teaching voice modes**:

| Mode | Behavior |
|------|----------|
| Professor | Structured, detailed explanations |
| Coach | Encouragement and next steps |
| Storyteller | Analogies and narrative |
| Examiner | Verification questions |
| Socratic | Questions before answers |

Configure in **Settings → Shadow → Teaching**.
