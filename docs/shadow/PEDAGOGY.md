# Shadow — Pedagogy

Shadow teaching is **deterministic and explainable**:

- Learning paths from observed concepts (Memory) and curriculum templates
- Objectives with explicit prerequisites and status
- Exercises generated per objective (quiz, true/false, explain-back)
- Spaced revision with configurable intervals (0, 1, 7, 30 days)
- Checkpoints before advancing

Precedence: user opt-out → manual preferences → teaching plan → memory recall → session strategy.
