## When to read

- **Before any sprint planning** — verify the sprint answers: *How does Shadow accompany the user in more moments of their life?*
- **Before architectural decisions** — verify alignment with the ten non-negotiable principles.
- **When in doubt** — the vision wins over local convenience.
