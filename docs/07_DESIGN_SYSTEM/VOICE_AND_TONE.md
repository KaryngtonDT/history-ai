# Brand Voice

History AI speaks like a **patient scholar** — clear, calm, precise.

Not a hype AI assistant. Not a corporate manual.

---

# Tone Attributes

| Attribute | Means | Avoid |
| --------- | ----- | ----- |
| Calm | Short sentences, no urgency theater | "Act now!", "Amazing!" |
| Focused | One idea per message | Wall of text |
| Scholarly | Respect for knowledge | Slang, memes, emoji in UI copy |

---

# UI Copy Patterns

## Dashboard tagline

> Transform knowledge into understanding.

## Empty states

Lead with possibility, not failure:

* ✅ "No content yet. Import your first document."
* ❌ "You haven't uploaded anything."

## Processing

* ✅ "Processing your document…"
* ❌ "AI is working magic on your file!"

## Errors

Structure: **What happened** + **What to do**

* "This file is too large. Maximum size is 50 MB. Choose a smaller PDF."

# Terminology

| Use | Avoid |
| --- | ----- |
| Import | Upload (either OK — pick one in MVP; prefer **Import** for user-facing) |
| Content | File, asset |
| Processing | AI magic, thinking |
| Library | Database, repository |

# Languages

Product copy is authored in **English** first.

Translation Features (Feature-0006+) apply to **learning content** — not UI chrome (initially).
