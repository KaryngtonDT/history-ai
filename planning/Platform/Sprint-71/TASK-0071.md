## Problem

The runtime catalog covered only the video pipeline (18 engines). Strategic direction adds:

- CPU/lightweight alternatives (Whisper.cpp, Piper)
- Lip sync restructuring (LivePortrait, MuseTalk legacy)
- New capabilities: OCR, Vision, Embeddings, Reranking

Users need a single maturity view: default → alternatives → hardware → provider → benchmark model.
