# Feature 0001 — PDF Upload

## Goal

Allow a user to upload a PDF document and start the processing pipeline.

## Business Value

Transforms a static PDF into an AI-ready learning resource.

## Actors

- Authenticated User

## Success Criteria

- PDF uploaded successfully
- Metadata stored
- Content created
- ProcessingJob created
- User sees processing status

## Out of Scope

- AI processing
- Summary
- Translation
- Quiz
- Podcast
