# TASK-0003 — Symfony Backend Foundation

## Milestone

M1 — Project Foundation

## Sprint

Sprint-00

## Status

Planned

---

# Objective

Create the Symfony backend foundation with a reproducible Docker build.

---

# Requirements

* Symfony 7
* PHP 8.4
* Doctrine ORM
* Messenger
* Serializer
* Validator

---

# Implement

* `/health` endpoint returning `{"status": "ok"}`
* Functional test for `/health`
* DDD layer folders under `backend/src/`

---

# Constraints

Do not install:

* API Platform
* Security Bundle
* JWT
* Twig
* UX packages
* Mercure
* Maker Bundle (production dependency)

Do not migrate to `apps/backend/` — keep `backend/` until Milestone 2 (ADR-0005).

No business logic. No database entities. Only HealthController.

Symfony must be installed via Docker build (reproducible), not manual local commands.

---

# Acceptance Criteria

* Symfony starts inside Docker.
* `GET /health` returns `{"status": "ok"}`
* Tests pass (`php bin/phpunit`).
* Docker healthcheck targets `/health`.

---

# Definition of Done

Backend foundation ready for TASK-0004.
