# TASK-0005 — Python Worker Foundation

## Objective

Create the Python worker foundation.

No AI logic.

No transcription.

No translation.

No queue integration yet.

---

## Requirements

Use:

- Python 3.13
- FastAPI
- uv
- Pydantic
- Ruff
- Pytest

---

## Deliverables

Create the worker application inside:

`worker/`

The worker must expose:

`GET /health`

Response:

```json
{
  "status": "ok",
  "service": "worker"
}
```

---

## Required structure

```text
worker/
├── app/
│   ├── api/
│   │   └── health.py
│   ├── core/
│   ├── services/
│   ├── workers/
│   ├── main.py
│   └── __init__.py
├── tests/
│   └── test_health.py
├── pyproject.toml
├── uv.lock
└── README.md
```

---

## Acceptance Criteria

- `docker compose up` starts the worker.
- `http://localhost:8001/health` returns status ok.
- `pytest` succeeds.
- `ruff check` succeeds.
- No AI processing is implemented.
- No business logic is implemented.
