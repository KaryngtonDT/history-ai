## Sprint rule

> **1 Sprint = 1 clear product objective.**

**Sprint 70 objective:** Shadow becomes a **Mobile Companion** with **official Personal Remote Access** — first Shadow client truly usable in daily life away from home, without public cloud exposure.

**Principle:** *One Shadow. One Home. Everywhere.*

**Not in this sprint:** Android Auto, WearOS, Apple Watch, advanced widgets, full offline mode, public cloud deployment, IDE companion (deferred to Sprint 71).

---

# Vision

The phone is a **light client**: display, voice I/O, commands. All intelligence runs on the **home Lumen server** (Docker prod-like, local storage, local models, Second Brain).

Desktop, Browser, and Mobile share:

- one Shadow identity
- one memory
- one Second Brain
- one conversation continuity

**Recommended transport for personal use:** [Tailscale](https://tailscale.com/) — no open ports, no public reverse proxy, no NAT configuration.

# Target architecture

```text
                 Shadow Clients

Desktop · Browser · Mobile (Flutter)

                        │
                        ▼
              Shadow Presence + Context Hub
                        │
                        ▼
              Transport Layer (opaque to Shadow)
         Localhost · LAN · Tailscale · Cloud (future)
                        │
                        ▼
              Lumen Platform (Docker @ home)
         Storage · Models · Worker · Second Brain
```

Shadow clients **must not** embed business logic duplicated from backend bounded contexts.

# Deployment profiles (official)

| Profile | Audience | Hosting | Transport |
| ------- | -------- | ------- | --------- |
| **Personal Local** | Developer / solo | Docker + localhost | `127.0.0.1` |
| **Personal Remote** ⭐ | Solo (recommended) | Docker + home PC | **Tailscale** |
| **Home Server** | Family / small team | Docker + NAS | Tailscale |
| **Cloud** | Public API (future) | K8s / managed | HTTPS public |

Sprint 70 implements **Personal Local**, **LAN**, **Tailscale**, and **Auto** (home Wi‑Fi → LAN, else → Tailscale). Cloud profile is documented only.
