# TASK-0067 — Shadow Second Brain & Knowledge Workspace (Sprint 67)

## Phase

**Phase 7 — Daily friction & consultability**

## Status

**Complete**

## Strategic note

Sprint 67 adds **almost no new intelligence**. It makes S55–S66 **consultable** in one place.

Shadow already knows a lot — but knowledge is scattered across Memory, Teaching, Mentor, Goals, Executive, Relationship, Knowledge Graph, Adaptive Learning. The user never sees it as a **single navigable workspace**.

> Shadow ne répond plus seulement. **Il navigue.**

This is the highest ROI sprint for the next six months of daily use.

---

# Objective

Build a **personal knowledge encyclopedia**: organized, searchable, interconnected, explainable. Every video, PDF, audio, conversation, mission, and exercise enriches one workspace.

Shadow must be able to say:

- "You already know this concept."
- "This concept appears in 5 videos, 2 PDFs, and 8 conversations."
- "You learned this three months ago but never revised it."
- "This new video adds only 3 novel concepts."
- "Here is exactly where we discussed this."

---

# Philosophy shift

**Before (S55–S66)**

```text
Video → Memory → Knowledge Graph → (hidden in bounded contexts)
```

**After (S67)**

```text
Sources → Memory → Knowledge Graph → Second Brain → Knowledge Workspace → Shadow UI
```

# Architecture

```text
Sources (PDF / Video / Audio / YouTube)
            │
            ▼
Memory Timeline (S62)
            │
            ▼
Knowledge Graph (S64)
            │
            ▼
Second Brain (S67) ← aggregate only, no new ML
            │
 ┌──────────┼────────────┐
 ▼          ▼            ▼
Explorer   Search     Insights (Diff, Merge, Stats)
            │
            ▼
Shadow (watch, settings, search)
```

**Inputs (read-only):** Memory, Knowledge Graph, Teaching, Mentor, Executive, Goals, Relationship, Adaptive Learning, pipeline artefacts.

**Outputs:** `KnowledgeWorkspace` — entries, sources, insights, timeline, bookmarks, notes.

## P67-SLICE-02 — Knowledge Workspace engine

`Application/ShadowSecondBrain/`:

- `WorkspaceBuilder` (facade — like `ExecutiveCoordinator`)
- `KnowledgeAggregator` — pulls from all Shadow bounded contexts
- `DuplicateConceptResolver`, `KnowledgeMergeEngine`
- `KnowledgeDiffEngine` — novel vs known vs revision for a resource
- `KnowledgeStatisticsBuilder`, `KnowledgeWorkspaceExplorer`
- `KnowledgeWorkspaceSearch`, `InsightGenerator`

Deterministic. No model training. No pipeline duplication.

---

## P67-SLICE-03 — Knowledge Diff

Before starting a video/PDF:

```text
New concepts: 8
Already known: 17
Revisions due: 4
Redundancy: low
```

Or: "This video covers 85% of what you already know."

API: `GET /api/shadow/brain/diff?resourceType=video&resourceId=…`

Watch UI: diff preview panel before play.
