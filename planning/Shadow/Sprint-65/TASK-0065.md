# Objective

Shadow becomes a **personal mentor**: it understands *why* the user learns, plans a transformation path, assigns missions, measures progress, and adapts ‚Äî always deterministically and with user validation.

---

# Philosophy shift

**Before (S55‚ÄìS64)**

```text
Question ‚Üí Memory ‚Üí Knowledge Graph ‚Üí Teaching ‚Üí Answer
```

**After (S65)**

```text
Career Goal ‚Üí Mentor Engine ‚Üí Knowledge Graph ‚Üí Teaching ‚Üí Learning Mission ‚Üí Conversation
```

The user no longer follows videos. They follow a **transformation journey**.

# Slices

## P65-SLICE-01 ‚Äî Goal domain

`Domain/ShadowGoals/`:

- `LearningGoal`, `CareerGoal`, `SkillGoal`
- `GoalCategory`, `GoalPriority`, `GoalStatus`
- `GoalMilestone`, `GoalConstraint`
- `ShadowGoalsRepositoryInterface`
- `Exception/InvalidShadowGoalException`

Categories: `career`, `language`, `programming`, `history`, `philosophy`, `university`, `certification`, `personal`, `custom`

Each goal: description, motivation, priority, deadline, target skills, required knowledge, success criteria, progress percent.

**Multi-goal (stretch P65-SLICE-09):** primary + secondary goals with orchestrated mission allocation.

---

## P65-SLICE-02 ‚Äî Mentor engine

`Application/ShadowMentor/`:

- `GoalPlanner`, `GoalAnalyzer`, `LearningMissionBuilder`
- `GoalProgressCalculator`, `GoalRecommendationEngine`
- `MilestoneResolver`, `GoalExplanationBuilder`, `MentorAdvisor`, `RoadmapBuilder`

**Inputs:** Knowledge Graph (S64), Memory (S62), Relationship (S61), Teaching (S63), Adaptive Learning, user preferences.

**Output:** `MentorPlan` (roadmap, current mission, milestones, skills radar, estimated completion).

Deterministic. No model training. Every recommendation explainable.

---

## P65-SLICE-03 ‚Äî Learning missions

New concept distinct from Teaching missions (S63) ‚Äî mentor-scoped, goal-linked.

Mission fields: title, objective, duration, prerequisites, exercises, validation, unlocked concept.

Wire completion ‚Üí `KnowledgeBuilder` + `TeachingBuilder` sync.

API: `POST /api/shadow/missions/{id}/complete`

---
